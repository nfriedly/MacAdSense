README
======

MacAdSense shows your AdSense earnings, click count, eCPM values for the
current month, and the extrapolated earnings for the whole month (by using a
very simple rule of three method) on your dashboard. 

Get more about this software infos at 
http://blog.oswaldism.de/macadsense-simple-adsense-widget-for-mac-os-x/
